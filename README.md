# 📱 만료알림 임대수첩 (Landlord Management App)

고령 임대사업자가 **계약 만료와 연락 일정을 놓치지 않도록 돕는 앱**입니다.
복잡한 기능보다 **“오늘 해야 할 일”을 한눈에 보여주고 바로 실행**하는 데 집중합니다.

---

## 🎯 문제 정의

기존 임대관리 앱은:

* 기능이 너무 많고 복잡함
* 세무/통계 중심이라 실제 관리 흐름과 맞지 않음
* 고령 사용자에게 사용 난이도가 높음

👉 그래서 이 앱은 다음에 집중합니다:

* “지금 뭐 해야 하지?”를 바로 보여주기
* 연락 / 일정 / 만료 관리 중심
* 실수 방지 UX

---

## 👤 대상 사용자

* 50~70대 임대사업자
* 5~15채 정도 직접 관리하는 사용자
* 스마트폰 사용은 가능하지만 복잡한 앱은 어려운 사용자

---

## 🚀 핵심 기능 (MVP)

### 1. 호실 관리

* 건물 / 호실 / 세입자 정보 관리
* 계약 종료일 관리

### 2. 계약 만료 관리

* 만료까지 남은 날짜 표시

  * N일 남음
  * 오늘 만료
  * N일 지남

### 3. 다음 연락일 관리

* 날짜 선택으로 일정 등록
* 수정 / 삭제 가능
* 상태(협의중)와 분리된 독립 기능

### 4. 오늘 할 일 대시보드

* 오늘 연락할 항목
* 협의중
* 이번 달 만료
* 만료 임박 TOP 3

👉 “앱 열면 바로 해야 할 일 보이기”

### 5. 로컬 알림

* 다음 연락일 알림
* 계약 만료 알림

  * 7일 전
  * 당일

---

## 🔁 사용자 흐름

1. 홈에서 오늘 할 일 확인
2. 호실 선택
3. 전화 또는 문자
4. 다음 연락일 설정
5. 필요 시 협의중 상태 변경
6. 이후 갱신 또는 종료 처리

👉 **연락 → 일정 → 상태 → 계약 처리** 흐름

---

## 🧱 화면 구조

### 1. 홈 화면

* 오늘 할 일 중심 대시보드

### 2. 호실 목록

* 전체 호실 리스트

### 3. 호실 상세

* 정보 요약 카드
* 빠른 작업 (전화 / 문자 / 일정)
* 계약 처리 (갱신 / 종료)
* 정보 관리 (수정 / 삭제)
